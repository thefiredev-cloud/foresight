# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability
**Do not open a public GitHub issue for security vulnerabilities.**

Email: security@thefiredev.com

We respond within 48 hours and provide a fix timeline within 7 days. 90-day coordinated disclosure window.
