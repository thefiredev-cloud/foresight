# Foresight OS

> Simulate Before You Commit

Run 10,000 scenarios on any business decision before you make it. See the futures. Pick the best one.

This landing page was built as part of The Fire Dev's moonshot product suite.

## Tech Stack

- Next.js 14 (App Router, static export)
- React 18
- Three.js via @react-three/fiber + @react-three/drei
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Outputs to `out/` directory (static export, ready for Netlify/Vercel).

## Features

- **Scenario Engine** — Generate thousands of realistic business scenarios
- **Market Simulation** — Model market conditions and competitor responses
- **Risk Quantification** — Get precise probability distributions for outcomes
- **Decision Confidence** — Walk into every meeting with certainty
